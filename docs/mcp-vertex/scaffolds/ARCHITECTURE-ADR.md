# Architecture: ADRs

## Purpose

Architecture decision records document one durable decision and its trade-offs.

## Required Shape

- `# NNNN — Title`
- `## Status`
- `## Context`
- `## Decision`
- `## Consequences`

## Validation

ADRs should be short, decision-focused, and linked from any proposal that
implements the decision.
