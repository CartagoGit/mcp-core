# Scaffolds

Canonical document-shape contracts for mcp-vertex files. Each scaffold
declares an `applies-to` glob and the headings/invariants a contributor or
agent should preserve when editing that file family.

## Index

- `ARCHITECTURE-PROPOSALS.md` — proposal frontmatter, folders, slices, and recovery.
- `ARCHITECTURE-AUDITS.md` — audit report evidence and findings shape.
- `ARCHITECTURE-ADR.md` — architecture decision records.
- `ARCHITECTURE-WORKFLOWS.md` — GitHub Actions workflow hygiene.
- `ARCHITECTURE-TOOLS.md` — MCP tool registration contracts.
- `ARCHITECTURE-DOCS.md` — web/product documentation pages.
- `ARCHITECTURE-MEMORY.md` — durable memory-note expectations.
